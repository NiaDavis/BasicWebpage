What's Front-End?

All the components of a website a user can see/ interact with. There are 3 main parts to the front-end.

HTML

Hyper Text Markup Language

The content of a webpage. (text, images, videos, etc.)

HTML is written in a series of tags .

CSS

Cascanding Style Sheets

The style of a webpage. (colors, borders, backgrounds, layout, spacing, etc.)

Usually CSS goes in another file, written as properties and values:

Example: tag/.className/#idName {property: value;}

Example: h1 {color: red;}

JavaScript

For interactivity - magic part for exciting websites.

Requires variables, functions, logic and an understanding of how webpages work and usually goes in another file.

There are a lot of libraries using JavaScript for less lines of code/ more effortless. Popular ones: jQuery, Angular and React.

CSS Basics

Colors - described as words (red), hex (#ff0000), rgb (rgb(255, 0, 0))

Font - for color, family, size, weight

Background - for color, images

Width and height - set the size of divs and images. For images, one can be set to auto so that the image scales correctly. Usually described using pixels (px), and percent (%).

Align - text-align used for text. (center, right, left). Images require a bit more work (see the example of the image in the HTML section - the size is described as a percent, display is set to block and the left and right margins are set to auto).

Display - used to specifiy where the content will be displayed. Example, inline-block means the content will be displayed side by side.

Border - described by color, size (pixels), type (solid, dashed).

Margin - space around the edge of content/ other content. Area outside the border. Can be specified for top, bottom, left and right.

Padding - space around the edge ofcontent/ other content. Inside the border of the HTML element. Can be specified for top, bottom, left and right.

Advanced HTML

Divs and Spans

Use extra HTML tags to help break up/ identify content.

  • Divs - stands for division. Split content into sections, and very useful for CSS and JavaScript.
  • Spans - apply CSS and JavaScript to specify content, (one word in a sentence).

Divs, paragraphs and headings are examples of block-level HTML tags. However, spans are inline - (a blank line is not taken before and after).

Note: CSS can be added to divs to ensure they don't take new lines.

Classes and IDs

Use special tags = classes and IDs to identify which tag/ group of tags to be specific.

  • Classes - describes a group of HTML tags to apply the same CSS or JavaScript to. Add "class=className" inside opening HTML tag. Referred to in CSS by ".className"
  • IDs - describes ONE HTML tag - only one with each ID name. Add "id=idName" inside opening HTML tag. Referred to in CSS by "#idName".

HTML tags can have both IDs and classe/ multiple classes.

Example: class="oneClass twoClass" id="specialID">

Class and ID names cannot begin with a number, spaces or special characters. Can include hyphens. Examples: camelCase and snake-case.
